Training and Tuning of Machine Learning Models for Alzheimers Disease Prediction

This notebook focuses on the training and evaluation of multiple machine learning models for Alzheimers disease prediction. Several classification algorithms are implemented and tuned, including Logistic Regression, tree-based models, ensemble methods, SVM, and KNN, in order to compare their predictive performance.

The models are trained under three different scenarios: without dimensionality reduction, with PCA retaining 95% of the variance, and with a 2D PCA representation mainly used for visualization. This allows analyzing the impact of dimensionality reduction on model performance and class separability.

Model performance is compared using standard evaluation metrics, and the best-performing models are saved and exported for later use in a separate notebook dedicated to making predictions on new data.

Components necessary for 95% variance: 27

Conclusion

Principal Component Analysis (PCA) was applied to the training dataset to reduce dimensionality and assess the amount of information retained by each component. The analysis showed that 27 components are required to capture 95% of the total variance, indicating that a large portion of the information is distributed across multiple variables and that it is not possible to drastically reduce dimensionality without losing significant information.

def entrenar_modelos(X_train, X_test, y_train, y_test):
    resultados = {}

    for nombre, (modelo, parametros) in modelos.items():
        print(f"\nTraining and optimizing  {nombre}...")

        grid = GridSearchCV(modelo, parametros, cv=5, scoring="f1_macro", n_jobs=-1)
        grid.fit(X_train, y_train)

        mejor_modelo = grid.best_estimator_
        y_pred = mejor_modelo.predict(X_test)

        resultados[nombre] = {
            "modelo": mejor_modelo,
            "accuracy": accuracy_score(y_test, y_pred),
            "f1": f1_score(y_test, y_pred, average="macro"),
            "precision": precision_score(y_test, y_pred, average="macro"),
            "recall": recall_score(y_test, y_pred, average="macro"),
            "matriz": confusion_matrix(y_test, y_pred),
        }

        print(f"Best Model: {mejor_modelo}")
        print("Accuracy:", resultados[nombre]["accuracy"])
        print("F1 Score:", resultados[nombre]["f1"])
        print("Precision:", resultados[nombre]["precision"])
        print("Recall:", resultados[nombre]["recall"])


    return resultados

def grafico_comparacion(metric, titulo, resultados_sinpca, resultados_pca95, resultados_pca2):
    modelos = list(resultados_sinpca.keys())

    sinpca = [resultados_sinpca[m][metric] for m in modelos]
    pca95 = [resultados_pca95[m][metric] for m in modelos]
    pca2 = [resultados_pca2[m][metric] for m in modelos]

    x = np.arange(len(modelos))
    width = 0.25

    plt.figure(figsize=(12,6))
    plt.bar(x - width, sinpca, width, label="Sin PCA")
    plt.bar(x, pca95, width, label="PCA 95%")
    plt.bar(x + width, pca2, width, label="PCA 2D")

    plt.xticks(x, modelos, rotation=45, ha="right")
    plt.title(titulo)
    plt.ylabel(metric.upper())
    plt.legend()
    plt.tight_layout()

    nombre_archivo = titulo.replace(" ", "_") + ".png"

    plt.show()


grafico_comparacion("accuracy",
                    "Accuracy Comparison (No PCA vs. PCA 95% vs. PCA 2D)",
                    resultados_sinpca, resultados_pca95, resultados_pca2)

Models without PCA:

  • Ensemble models (Gradient Boosting and XGBoost) showed the best performance, with F1-scores of 0.949 and 0.944, respectively, and accuracies above 94%.

  • Individual tree-based models (Decision Tree and Random Forest) also achieved strong performance (F1 0.920.93).

  • Linear and distance-based models (Logistic Regression, SVM, and KNN) obtained moderate results, with KNN showing the lowest performance (F1 0.71).

  • This indicates that the dataset and original variables allow ensemble models to better capture the complexity of the relationship between features and diagnosis.

Models with PCA 95% (95% variance retention):

  • Performance slightly decreased compared to models without PCA, but Gradient Boosting, XGBoost, and Random Forest remained the best-performing models (F1 0.810.82).

  • This shows that dimensionality reduction while retaining 95% of the variance allows good performance to be maintained, although slightly lower than when using all features.

Models with PCA 2D (only two principal components for visualization):

  • Performance deteriorated significantly (F1 0.530.59).

  • This aligns with the observation that the first two principal components do not clearly separate the classes, making classification more difficult.

Results

A dimensionality reduction analysis was performed using PCA, finding that 27 components are required to retain 95% of the variance, while the 2D projection showed that the classes overlap, indicating low separability. For the KNN model, the optimal number of neighbors was 19, and the Manhattan distance metric proved to be the most effective (F1 = 0.7435), followed by Euclidean and Chebyshev.

In training models without PCA, XGBoost achieved strong performance (Accuracy 94.88, F1 0.949), followed closely by Gradient Boosting (F1 0.944), while linear models and KNN showed lower performance. With PCA retaining 95% of the variance, performance decreased slightly, but Gradient Boosting and XGBoost remained the best-performing models. The confusion matrix of XGBoost showed few false positives (8) and false negatives (14), confirming that the model correctly identifies most patients with and without Alzheimers disease.

SMOTE was also applied; although it balanced the classes during training, in this case it did not improve the overall model performance, and the metrics even decreased slightly. This indicates that XGBoost already handled class imbalance well in this dataset, making the application of SMOTE unnecessary for improving prediction performance.

Conclusion

In conclusion, XGBoost proved to be the most effective model for predicting Alzheimers disease in this dataset, achieving high accuracy and F1-score even without applying class balancing. Dimensionality reduction using PCA with 95% variance retention preserved a substantial portion of the useful information, while the 2D projection showed low class separability. KNN was sensitive to the choice of distance metric, with Manhattan performing best. Overall, the results indicate that tree-based models are the most suitable for this task.
